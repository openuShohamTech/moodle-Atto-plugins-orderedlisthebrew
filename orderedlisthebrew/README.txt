ordered list in Hebrew Alef Bet for Moodle Atto

This is a plugin for the Moodle Atto.
It will display a button to create a Hebrew ordered list.

1. Download
============
Download the plugin from: https://github.com/openuShohamTech/moodle-Atto-plugins-orderedlisthebrew.git


2 Unzip / Rename / Upload
============
Expand (unzip) the zip file. 

Rename the main folder to "orderedlisthebrew" 
It should contain all the files like "version.php." 

Upload the "orderedlisthebrew" folder into the folder

[PATH TO MOODLE]/lib/editor/atto/plugins 


3. Get Moodle to Install It 
============
Visit Settings > Site Administration > Notifications, and let Moodle guide you through the install.


4. Configure it
============
Go to Site Administration > Plugins > Text Editors > Atto Toolbar Settings  

Now add orderedlisthebrew to the menu structure near the bottom of the page

  e.g style1 = title, bold, italic, orderedlisthebrew

  (where widget is the name of your atto plugin)

Thats all. The orderedlisthebrew icon, a big grey G. should now appear on the Atto HTML editor toolbar. 
When you click it, orderedlisthebrew will give you a ordered list in Hebrew of all the selected text.
